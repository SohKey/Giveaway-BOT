# Requirements

- Git   ```apt install git```
- Python >= 3   ```apt install python3-pip```
- Pip

## Install

```bash
$ git clone https://github.com/SohKey/Giveaway-BOT.git #clone repository
$ cd NFT-Giveaway-BOT
$ nano tools/config.ini #Edit account configuration and more !
```
Minimal configuration:
- Reddit id and pass 
- API name, id and secret 
- NFT Wallet

## Usage

```bash
$ python3 botr.py
```
