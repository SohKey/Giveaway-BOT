# Requirements

- Python >= 10
- Window 10 and more

## Install

```bash
$ git clone https://github.com/aresrpg/aresrpg
$ cd aresrpg
$ git submodule update --init
$ npm install
```

## Usage

```bash
$ npm start
# Start with packet logging
$ DEBUG="minecraft-protocol*" npm start
```
